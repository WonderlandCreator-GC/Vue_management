# Vue-Shop-Management-System
## 项目简介
  此项目为商场后台管理系统，基于Vue+elementUI开发，实现的功能有用户管理，权限管理，分类管理，参数管理，商品管理，数据报表统计。仓库内容包含前端部分源代码。
